# 🎓 Prediksi Mahasiswa Dropout

Aplikasi ini dibuat menggunakan **Streamlit** untuk memprediksi kemungkinan mahasiswa mengalami **Dropout** menggunakan algoritma **Random Forest Classifier**.

## 📊 Dataset

Students Dropout and Academic Success Dataset

## 🤖 Algoritma

- Random Forest Classifier

## 🛠️ Library

- Streamlit
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Joblib

## 🚀 Cara Menjalankan

1. Clone repository
2. Install library

```bash
pip install -r requirements.txt
```

3. Jalankan aplikasi

```bash
streamlit run app.py
```
